# After 30 Med Bot — Telegram бот для напоминаний о приёме лекарств

> ⚡ Вайб-кодинг проект — написан в паре с Claude (AI).
> Код живой, рабочий, итерируем дальше.

## Возможности

### 💊 Лекарства и расписание
- **Добавление лекарств** с дозировкой, способом приёма (до / после / во время еды / независимо) и гибким расписанием
- **Гибкое расписание**: каждый день · через N дней · по дням недели · раз в месяц
- **Multi-select время приёма** — 🌅 Утро / ☀️ Обед / 🌇 Вечер / 🌙 Ночь в любой комбинации
- **Разная дозировка** — одно лекарство с двумя дозировками и отдельным расписанием для каждой (например: 25 мкг по нечётным / 50 мкг по чётным)
- **Пауза лекарства** — временно отключить напоминания (например, в отпуске) без удаления, в один клик; на паузе не учитывается в соблюдении
- **Редактирование и удаление** в пару кликов

### ⏰ Напоминания
- Напоминания по расписанию с кнопками ✅ Принял / ❌ Пропустить
- **Два режима**: один раз или повтор каждые 5 минут до подтверждения (до 2 часов)
- **Персональные пресеты времени** — каждый настраивает своё время слотов; смена пресета переносит все существующие напоминания
- **План на день** — утреннее сообщение со списком лекарств на сегодня
- **Лекарства на сегодня** — расписание с отметками ✅ принято / ❌ пропущено / ⏳ ожидает

### 📦 Запас таблеток
- Указать остаток в упаковке и расход за приём
- Автосписание при подтверждении приёма
- Предупреждение, когда запас заканчивается (за N дней, настраивается)
- Прогноз «хватит примерно на N дней» для любого расписания

### 📊 Статистика и экспорт
- **История за 7 дней** с детализацией по дням, времени и статусу
- **Соблюдение за 30 дней** (adherence) — % принятых от положенных по расписанию, по каждому лекарству и итог, с индикатором 🟢/🟡/🔴
- **Серия идеальных дней** 🔥 — сколько дней подряд приняты все лекарства; отдельно для себя и каждого подопечного, с майлстоунами (⭐ 7 / 🏆 30 дней)
- **Отчёт для врача** — PDF-календарь приверженности за 30 дней (альбомная ориентация): дни раскрашены по соблюдению (🟩 ≥90% · 🟨 50–90% · 🟥 <50%), внутри — лекарства с цветным кружком (принято/пропущено); отдельная страница на каждого подопечного
- **План на 7 дней** по датам с учётом типа расписания
- **Экспорт в PDF** — историю, план, отчёт о соблюдении и отчёт для врача (кириллица через DejaVuSans TTF)

### 👨‍👩‍👧 Caregiver-режим
- Отслеживание приёма лекарств для близких (до 2 подопечных)
- При добавлении лекарства бот спрашивает «Для кого?»
- Напоминания и история с пометкой подопечного

### ⚙️ Прочее
- **Единое меню `/menu`** с навигацией «на месте» и кнопкой «◀️ В меню» на всех экранах
- **Определение часового пояса** по геолокации или городу
- **Несколько пользователей**, лимит 10 лекарств на пользователя/подопечного
- **Удаление данных** — полное удаление всех своих данных из /settings

## Стек

- Python 3.14
- python-telegram-bot 22.7 — Telegram API (async)
- APScheduler 3.11.2 — планировщик напоминаний (каждую минуту)
- SQLite — база данных
- pytz + timezonefinder + geopy — определение часового пояса
- fpdf2 + DejaVuSans TTF — генерация PDF
- pytest — тесты (`requirements-dev.txt`)

## Структура проекта

```
med-bot/
├── bot.py              # точка входа — регистрация handlers, main()
├── database.py         # вся работа с SQLite (CRUD + миграции)
├── scheduler.py        # отправка напоминаний каждую минуту
├── schedule_utils.py   # чистая логика «положенных приёмов» + прогноз запаса
├── streak.py           # серия идеальных дней (по себе и подопечным)
├── constants.py        # состояния диалогов, метки, слоты времени
├── utils.py            # общие утилиты (декоратор ошибок, escape_md/html, cancel)
├── broadcast.py        # скрипт рассылки сообщений всем пользователям
├── handlers/
│   ├── meds.py         # добавление, редактирование, удаление лекарств
│   ├── stock.py        # экран «📦 Запас» (остаток, расход, порог, прогноз)
│   ├── stats.py        # статистика за 7 дней, план на 7 дней
│   ├── export.py       # экспорт истории и плана в PDF
│   ├── settings.py     # настройки, пресеты времени, режим напоминаний
│   ├── caregiver.py    # caregiver-режим, подопечные
│   ├── admin.py        # админ-панель (только для ADMIN_ID)
│   └── timezone.py     # /start, часовой пояс, главное меню /menu
├── tests/              # pytest (чистые функции, БД, хендлеры)
├── .env                # токен бота (не коммитить!)
├── requirements.txt
└── med_bot.db          # SQLite база (создаётся автоматически)
```

## База данных

| Таблица | Описание |
|---|---|
| `users` | пользователи: telegram_id, timezone, reminder_mode, пресеты времени, план дня, caregiver |
| `dependents` | подопечные caregiver-режима |
| `medications` | лекарства: название, дозировка, способ приёма, кол-во приёмов, запас (F5) |
| `schedule_rules` | правила расписания: время, тип, `dosage` (NULL = берётся из medications) |
| `intake_log` | история приёмов (taken / skipped / pending) |

### Типы расписания

| Тип | Описание |
|---|---|
| `daily` | каждый день |
| `interval` | каждые N дней от даты начала |
| `weekdays` | по выбранным дням недели (Пн–Вс) |
| `monthly` | раз в месяц, N-го числа |

## Команды бота

| Команда | Описание |
|---|---|
| `/menu` | единая точка входа — главное меню (навигация по всем разделам) |
| `/start` | регистрация, запрос часового пояса |
| `/cancel` | отмена текущего диалога |

> Из меню доступны: Лекарства на сегодня · Мои лекарства · Статистика · Настройки · О проекте.

```bash
source venv/bin/activate

```
